My Projects

Portfolio Website

A fully responsive and modern portfolio website showcasing personal skills, featured projects, testimonials, and a contact form. It uses smooth scroll animations, subtle parallax effects, and dark/light mode toggling for a professional touch. Built with HTML, CSS, and JavaScript.

E-commerce Store

An interactive online store with a real-time shopping cart, user login system, and product filters by category, price, and rating. It features a backend integration for inventory management and Stripe API for secure checkout, offering a seamless shopping experience across all devices.

Crowdfunding Platform

A web-based crowdfunding application where users can create and support fundraising campaigns. Each campaign includes a story, goal amount, progress bar, and donation history. It includes user authentication, profile dashboards, and integrates payment handling for contributors.

Recipe Finder App

A beautifully designed recipe search app that fetches meals based on ingredients or keywords using an external API. Users can view full instructions, ingredient lists, and calorie breakdowns. The app is fully responsive, with saved favorites and filtering by meal types like vegan or gluten-free.

Blog Website

A personal blogging platform allowing users to create, edit, and publish blog posts with markdown support. Features include category-based filtering, search functionality, pagination, and a minimalist user interface that adapts well to mobile and desktop screens. Includes dark mode and comment section.

Weather Dashboard

A real-time weather forecast dashboard using the OpenWeatherMap API. Users can search for any city and view current weather conditions, 5-day forecasts, humidity levels, wind speed, and UV index, all with intuitive weather icons and animated backgrounds that reflect the forecast.

Movie Search Engine

A dynamic web app where users can search for movies by title or genre using the OMDB API. Displays movie posters, release years, summaries, and ratings. Users can add movies to their favorites, and results are shown in a stylish grid layout with responsive design.

Task Management App

A to-do and task organizer with drag-and-drop Kanban boards. Users can create projects, assign tasks, set due dates, and track progress using statuses like “To Do,” “In Progress,” and “Done.” Built with JavaScript and local storage, and styled with a clean UI for productivity.

Travel Landing Page

A landing page for a travel company, featuring beautiful hero sections, animated destination cards, customer testimonials, and a booking form. Built with responsive design principles, it uses scroll-triggered animations and creative image layering for a premium visual experience.

Quiz Game App

An interactive quiz game with multiple-choice questions and instant feedback. Tracks user scores, includes a countdown timer, and provides final results with suggestions for improvement. Designed with transitions between questions and color indicators for correct/incorrect answers.

Music Player UI

A sleek music player interface featuring custom play, pause, next, and volume controls. Includes a song list, animated progress bar, and album art. Built using JavaScript to handle audio functions and CSS transitions for elegant visual feedback.

Login & Signup Authentication UI

A modern authentication interface with animated form transitions for login, sign-up, and password reset. It features field validation, show/hide password toggles, and error alerts. Built with accessibility in mind and styled with a clean minimalist layout.
